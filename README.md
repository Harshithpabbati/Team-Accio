# Mail Room Service

### Installing
You need to have the latest version of Nodejs and mongodb.
Install the required npm packages

```
npm install
```

Run the demo

```
npm start
```

## Built With

* [Node js](https://nodejs.org/en/) - The web framework.
* [Boot strap](https://getbootstrap.com/) -UI
